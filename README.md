— Downloads & Installation

⚪ writenode |... is a voice-powered cognitive layer for your browser. It lives in Chrome as an extension — persistent notes, AI chat with memory, a spatial canvas, and a knowledge graph that connects everything you think about.

---

## Download the Latest Release

1. Look for the **"Releases"** section on the **right side** of this GitHub page
2. Click the latest release (e.g., **v0.6**)
3. Under **Assets**, click the `.zip` file to download it (e.g., `writenode-extension-v0.6.zip`)
4. Unzip the downloaded file — you'll get a folder with the extension files inside

---

## Install in Chrome

1. Open Chrome and go to **`chrome://extensions`** (type it directly in the address bar)
2. Turn on **Developer mode** — it's the toggle in the **top-right corner** of the page
3. Click **"Load unpacked"** — it's the button in the **top-left corner**
4. Select the **unzipped folder** you downloaded (the one containing `manifest.json`)
5. WriteNode will appear in your extensions list with a "digital pen" icon

That's it — you're running ⚪ writenode |...

---

## After Installation

- **Click the "digital pen" icon** in your Chrome toolbar to open the popup (sign in, settings, notes)
- **Look for the floating button** on any webpage — it's your quick-access menu for notes, canvas, voice, and the knowledge graph
- **Pin the extension** by clicking the puzzle piece icon in Chrome's toolbar, then clicking the pin next to ⚪ writenode |...

---

## Updating to a New Version

1. Download the new release `.zip` from this page
2. Unzip it (replace the old folder or use a new one)
3. Go to `chrome://extensions`
4. Click the **refresh icon** on the WriteNode card, or click **"Remove"** and then **"Load unpacked"** again with the new folder
